Viewport window

the Viewport window as the name suggest it the part of the editor where you acculity se what you create and has to mane configurations; The Scene View that displays the world from the creators view and the Gamer view who display what the acuall gamer vill see.

Scene view - tools

  1. Hand Tool ❨Q❩ - move across your project by dragging the scene around.
  2. Move Tool ❨W❩ - press an object and get arrows in X, Y and Z axis and move the object in respective direction by draging in the arrows
  3. Rotate Tool ❨E❩ - you press a objects and X, Y, Z orbits will be shown who cou can drag to change the rotation of marked object
  4. Scale Tool ❨R❩ - works like the move tool, you press a objects and X, Y, Z arrows but when you drag an axis it will scale the object in respective directions
  5. Rect Tool ❨T❩ - a tool that combines the move, rotate and scale tool idle for 2D use
  6. Transform Tool ❨T❩ - works like rect tool and combine the move, rotate and scale tool but is idle for 3D use
  7. Display modes - change specificaly what containt you want to focus at, you could get a mesh formate of your world, you could use it to easyer see and work with lightning etc.
  8. 2D/3D - toggle between 2D and 3D view
  9. Lights on/off - on shows in game lights with shadows and off for easy building enviroment
  10. Audio on/off
  11. Fog effects on/off makes you able turn off fog for easyer development
  12. Camera settings, camera with, speed etc

Scene view - how to move

  • Move around by holding down right mouse button and navigation with WASD with aditional QE to go up and down (you can speed up your movement by holding down shift button)
  • Drag around by holding down scroll wheel on mouse while moving the mouse
  • Zoom in/out by using the scrollwheel or moving two fingers towards and against each other axross the mousepad
  • Focus on object by selectiong an object and press F the program will automaticly give you a good perspective of the object
