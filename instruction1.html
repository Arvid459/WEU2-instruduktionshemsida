Unity Setup

Install Unity Hub

1. Navigate to Unitys oficial website to download Unity Hub

2. Press on the verion to the operating system that you currently are running

3. When the file it fully downloaded you will se it in the up-right corner of your browser and you will press it to run the installer

Create a Unity Account

1. If you dont alredy have a Unityacount you need to create one but if you alredy have one its just to log in

Install Unity Editor

1. In Unity Hub go to the Installs tab and than press on "Install Editor

2. Choose version, i prefer the latest version becuse of it... but you can also chose an older version for fever buggs

3. Press install

Create a new prodject

  1. Go to "Projects" in Unity Hub
  2. Press "New project"
  3. Chose "All Templates" to see all templates
  4. Chose what kind of project you want to make
  5. Edit project name and if you want to change the location
