User

In this chapter you will be learnd how to create an basic 3D user in unity with moving bodybarts, controlls and a camer that will follow the user

Physical body

  1. Right click on the Hierarchy window
  2. go to 3D-object and creathe a Capsule as the main body
  3. Repeatenthis steps with cylinders as arms and leggs and a creathe as a head
  4. In the Hierarchy window press right key and create an "Empty GameObject" and name it player
  5. move all body parts into the player gameobject by tragin them into the folder

In the inspector window you get a few option to use when styling your enviroment

Movement

To add movement logic you just add another conponent with the type script and in that code document you inplement the following code 


using UnityEngine;

public class SimplePlayerController : MonoBehaviour
{
    public float moveSpeed = 5f;
    public float rotationSpeed = 720f;

    void Update()
    {
        float h = Input.GetAxis("Horizontal");
        float v = Input.GetAxis("Vertical");

        Vector3 direction = new Vector3(h, 0, v).normalized;

        if (direction.magnitude >= 0.1f)
        {
            // Move the player
            transform.Translate(direction * moveSpeed * Time.deltaTime, Space.World);

            // Rotate the player
            Quaternion toRotation = Quaternion.LookRotation(direction, Vector3.up);
            transform.rotation = Quaternion.RotateTowards(transform.rotation, toRotation, rotationSpeed * Time.deltaTime);
        }
    }
}

Camera

follow the following steps to make a camera that follows any object, in this case the user

  1. Navigate into the player folder
  2. right click and press camera, make sure that the camera is being created inside the player folder otherwise it wount work
  3. line up the camera by draging in the Y, Z and X axis-arrows and see the result in the camera view below
  4. lastly if you want to start the game in player view make sure to chang the tag to "MainCamera
